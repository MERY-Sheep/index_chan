## ドキュメント管理方針

### 更新ルール

**アクティブなドキュメント**
- MVP/配下の主要ドキュメント
- 調査/配下の最新調査結果
- 常に最新の状態を保つ

**アーカイブ**
- MVP/archive/配下
- 過去の進捗報告
- 参考資料として保持

### 命名規則

**進捗報告**
- `Phase{番号}_{内容}.md`
- 例: `Phase1.5_LLM統合.md`

**完了報告**
- `Phase{番号}_最終報告.md`
- 例: `Phase1.5_最終報告.md`

**調査結果**
- `{テーマ}_実機テスト結果.md`
- `{テーマ}_参考プロジェクト分析.md`
